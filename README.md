# 🛡️ Ami-Security-Bot
هـادا بـوت مـصـاوب بـالـبـايـثـون لـحـمـايـة الـحـسابات والـتـعـامـل مـع الـأقـنـعـة (User-Agents).

## 🛠️ الـمـمـيـزات:
* تـغـيـيـر الـقـناع بـشـكـل عـشـوائـي (Random User-Agent).
* فـحـص الـمـواقـع بـاسـتـخـدام مـكـتـبـة Requests.
* نـظـام لـلـتـجـربة والـتـأكـد مـن الـحـالـة (Status Code).

## 🚀 كـيـفـاش تـخـدّم الـبـوت:
1. نـزل الـكـود (Clone).
2. نـزل الـأدوات: `pip install requests`.
3. خـدّم الـمـلـف: `python main.py`.
